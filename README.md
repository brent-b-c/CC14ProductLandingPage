# Product Landing Page

This project is a responsive single page product landing page built using HTML5, CSS3, and JavaScript. It showcase a sample product of the Toyota Fortuner.

## Features
- Semantic HTML5 structure
- Responsive design using CSS3 and media Queries
- Flexbox and CSS grid Layout
- Interactable mobile menu
- Interactive buttons for product actions

## Technologies used 
- HTML5
- CSS3
- JavaScript
- Git and Github
